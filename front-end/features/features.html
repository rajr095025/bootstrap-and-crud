Features of our website

Feature 1

Add User

We can add user to the database using this feature. There will be a tab available in Navigation Bar called "Add-user" which will help us add user firstname, lastname, email, phone number to the database.

Feature 2

Get All User

We can view all the users whom was added to the database using this feature. There will be a tab available in Navigation Bar called "Users-List" which will help us view users firstname, lastname, email, phone number present in the database.

Feature 3

Update a user

We can update any selected user who was already exist in the database using this feature. We can do this in two ways.
i) There will be a tab available in Navigation Bar called "Update-User" which will help us update user any details(firstname, lastname, email, phone number) using email. The changes we give will update that user details in database.
ii) We can also update user by seeing the user list and there will be a button available. When we click that button, we get a modal which will show the user details we can change the details and give submit to apply changes in database.

Feature 4

Delete a User

We can delete any selected user who was already exist in the database using this feature. We can do this in two ways.
i) There will be a tab available in Navigation Bar called "Delete-User" which will help us delete particular user using email. The user will be deleted from database.
ii) We can also delete user by seeing the user list and there will be a button available. When we click that button, we can delete that particular user and this will remove that user from database.
